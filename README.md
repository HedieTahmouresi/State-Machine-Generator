# 🎭 State Machine Generator

> A sleek Python tool that parses regular grammars and visually brings their finite state machines to life.

---

### 📌 Overview
Developed for a **Theory of Languages and Automata** course, this tool accepts a grammar in EBNF format and validates its regularity. 

* **Success Path:** Automatically maps out the grammar, rendering a visual graph (`.png`) alongside a detailed structural report (`.txt`) analyzing states, transitions, dead states, and determinism.
* **Error Path:** Safely catches non-regular syntax or structural faults, halting execution with a clean error breakdown.
